# Claude_Python_Testbed
A repo containing various Python scripts written using Claude Code.

## Contents

- **app.py** — A basic tkinter GUI application that displays a main window with a "Hello, World!!" label.
- **CLAUDE.md** — Project instructions and conventions for Claude Code sessions.

## Running

Requires Python 3 with tkinter (included in standard library).

```bash
py app.py
```
